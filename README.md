Как работает эта программа??
1. Сначала вводится команда в терминал с указанием того что вы хотите парсить.
Пример: cd product_analytics1
        python manage.py parse_products телефон
Затем запускаете django, командой: python manage.py runserver
Теперь уже можно запустить фронтенд, делается это так:
cd product_frontend/my-react
npm start
Всё, вы увидите все спарсенные товары и соответсвующие им графики.
В идеале можно было бы сделать так чтоб команду дял парсинга можно было бы отправлять, на месте через фронт, запрос шел бы в апи, оттуда в парсер, парсер в свою очередь собрал бы данные с вб, отправил все обратной дорогой. Но в ТЗ этого не уточнялось, а программа работает как надо.
